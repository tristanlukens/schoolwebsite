Over mij

Ik ben Tristan, 15 jaar en houd van drummen en programmeren.

Maar dat is wel heel kort samengevat.

Over mijzelf

Op 8 juni 2006 werd ik in Stadskanaal geboren, en ik woon al mijn hele leven in Gasselternijveen. Ik ga naar school in Stadskanaal, naar het Ubbo Emmius, aan de Stationslaan. Ik doe daar HAVO met een NT-profiel, met informatica (zoals waarschijnlijk wel duidelijk is) en muziek.

Programmeren

Ik doe voornamelijk web development, meestal met Svelte(Kit) en TailwindCSS. Omdat ik goed ben in de UNIX-terminal (zsh, lijkt veel op bash), initialiseer ik mijn project als volgt.

mkdir project
cd project
npm init svelte@next
npx svelte-add@latest tailwindcss
npm install

Dit gaat zeker veranderen, maar dit is zo op het moment van schrijven (11 november 2021).

Ik vind de syntax van React en Vue nogal vervelend, en ik vind Svelte gewoon ontzettend fijn om mee te werken. Blokjes HTML, CSS en JS (of eigenlijk HTML, TailwindCSS en TS) vind ik veel logischer dan gewoon. React en Vue zijn ook goede frameworks, echter zijn ze wat ingewikkelder. Ook zijn ze wat langzamer (door de virtual DOM) dan Svelte (een compiler).

Ik doe ook nog weleens wat met go, maar meer web dev. Ik probeer nu TS echt goed onder de knie te krijgen.

Drummen (en muziek in het algemeen)

Ik houd van drummen. Het is, naast programmeren het leukste wat ik kan doen. Ik drum het meest rock en metal, waar ik ook het meest naar luister. Ik heb een Roland TD-9 KX (niet mijn foto); ik drum elektronisch.

Op het moment luister ik veel naar de volgende bands.

Zoals je kunt zien zit ik weer een rockfase. Ik denk dat dit deel van deze pagina veel gaat veranderen als ik de moeite ervoor wil nemen.

Just to name a few.

Sport

Ik doe niet echt aan sport. Ik fiets natuurlijk wel iedere weekdag van huis naar school en weer terug, wat (enkele reis) ongeveer 8 kilometer is. Ik heb een tijd freerunning gedaan, maar na een jaar of drie (volgens mij) gestopt omdat mijn trainer ermee stopte.
